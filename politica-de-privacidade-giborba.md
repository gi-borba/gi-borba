## 2. Quais Dados Coletamos

Coletamos apenas os dados estritamente necessários para a prestação do serviço:

### 2.1 Dados fornecidos diretamente por você
- **Nome completo:** para identificação e personalização do atendimento
- **Número de WhatsApp:** para agendamento, comunicação sobre aulas e suporte
- **Outras informações que você nos enviar** voluntariamente via WhatsApp, formulário de contato ou Instagram Direct (ex.: histórico de saúde relevante para a prática de yoga, quando compartilhado por iniciativa própria)

### 2.2 Dados coletados automaticamente (site)
- Dados de navegação básicos (cookies de funcionamento e análise), quando aplicável à plataforma usada para hospedar o site
- Esses dados são anônimos e utilizados apenas para entender o desempenho do site

### 2.3 Dados coletados via plataformas de terceiros
- **Zoom:** ao participar das aulas ao vivo, a plataforma Zoom pode coletar dados técnicos (IP, dispositivo, duração da conexão). Esses dados são regidos pela [Política de Privacidade do Zoom](https://explore.zoom.us/pt/privacy/).
- **Hotmart:** ao acessar as gravações das aulas, você utiliza a plataforma Hotmart, que coleta dados de cadastro (nome, e-mail) e dados de acesso necessários à criação da conta de membro. O tratamento desses dados é regido pela [Política de Privacidade da Hotmart](https://hotmart.com/pt-BR/privacy-policy). A Hotmart também pode atuar como processadora de pagamentos. Nesse caso, os dados de pagamento são tratados exclusivamente por ela, sem acesso de Giovana Borba a dados de cartão ou dados financeiros completos.
- **Instagram / YouTube:** interações nessas plataformas seguem as políticas de privacidade do Meta e do Google, respectivamente.

## 3. Para Que Usamos Seus Dados

Seus dados são utilizados exclusivamente para:

| Finalidade | Base Legal (LGPD) |
|---|---|
| Agendamento e confirmação de aulas | Execução de contrato (Art. 7º, V) |
| Comunicação sobre horários, temas semanais e avisos | Execução de contrato (Art. 7º, V) |
| Envio de link do Zoom para as aulas | Execução de contrato (Art. 7º, V) |
| Responder dúvidas e oferecer suporte | Legítimo interesse (Art. 7º, IX) |
| Envio de informações sobre novos planos ou retiros Inspire Trip | Legítimo interesse / Consentimento (Art. 7º, I e IX) |
| Personalização da prática (considerações de saúde informadas) | Consentimento explícito (Art. 7º, I) |

**Não usamos seus dados para:**
- Venda ou compartilhamento com terceiros para fins comerciais
- Marketing de empresas parceiras
- Criação de perfis automatizados para tomada de decisões

## 4. Com Quem Compartilhamos Seus Dados

Seus dados **não são vendidos, alugados ou compartilhados comercialmente** com terceiros.

Podemos compartilhar dados apenas nas seguintes situações:

- **Plataformas operacionais necessárias ao serviço:** Zoom (para realização das aulas ao vivo), Hotmart (para disponibilização das gravações, área de membros e processamento de pagamentos), WhatsApp (para comunicação), plataforma de hospedagem do site. Esses fornecedores são contratados para viabilizar o serviço e atuam como operadores sob nossas instruções ou, no caso da Hotmart, como controladores independentes para os dados que processam diretamente em sua plataforma.
- **Obrigação legal:** quando exigido por lei, ordem judicial ou autoridade competente.

| Tipo de dado | Prazo de retenção |
|---|---|
| Dados de contato (nome + WhatsApp) de alunas ativas | Enquanto durar a relação de serviço |
| Dados de contato de alunas inativas | Até 12 meses após o último contato |

Após os prazos acima, os dados são excluídos ou anonimizados.

## 6. Seus Direitos como Titular dos Dados

A LGPD garante a você os seguintes direitos, que podem ser exercidos a qualquer momento pelo WhatsApp **+55 11 98753-0304**:

- **Acesso:** saber quais dados temos sobre você

Respondemos a solicitações de direitos em até **15 dias úteis**.

## 7. Segurança dos Dados

Adotamos as seguintes medidas para proteger seus dados:

- Comunicações realizadas por canais criptografados (WhatsApp, Zoom)
- Acesso aos dados restrito à professora Giovana Borba
- Dados não armazenados em planilhas públicas ou sistemas sem controle de acesso
- Gravações das aulas disponíveis apenas para alunas cadastradas, por link privado e temporário

Em caso de incidente de segurança que possa causar risco ou dano a você, comunicaremos dentro do prazo exigido pela LGPD e pelas orientações da ANPD.

Não utilizamos cookies para publicidade comportamental ou rastreamento entre sites.
